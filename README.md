## Website de Previsão do tempo

Projeto de estudo em *Front End* e *User Interface*, o qual englobou o desenvolvimento de uma interface destinada a exibir a temperatura atual do clima, índices de umidade, precipitação e velocidade do vento.

Para construir sua base, utilizei como guia o vídeo do canal <a href="https://www.youtube.com/watch?v=qC_DtdhH-J4&t=193s">*'Minimal Weather App Ui Design'*</a>, e para personalização, implementei as seguintes modificações:

(1) Tradução de textos para o português. <br>
(2) Alteração do plano de fundo e da imagem principal para fotos da cidade do Recife. <br>
(3) Implementação da previsão do tempo para mais dias da semana. <br>
(4) Incorporação de uma caixa de mensagem ao entrar no website. <br>
(5) Adaptação da página para ser responsiva em diferentes dispositivos.
